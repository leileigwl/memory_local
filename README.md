# Memory Local

> **你的 AI 第二大脑。它记得一切，永不遗忘。**

---

## 问题

你和 Claude 聊了几百次。

每次新对话，你都要重新告诉它：
- 你的项目背景
- 你的工作习惯
- 你的语言风格
- 你上周做过什么决定

**AI 有记忆，但记忆不持久。**

对话一结束，一切归零。

---

## 解决方案

Memory Local 是一个 Claude Code Skill。

它做一件事：**让 AI 记住你。**

不是记住一两条信息，是记住你的**思维方式**。

---

## 它怎么工作？

```
你说「记录」
    ↓
AI 扫描当前对话 + 所有历史 session
    ↓
自动提取：
  - 你的思维模式
  - 你的工作习惯
  - 你纠正过的错误
    ↓
写入本地偏好文件
    ↓
下次对话自动加载
    ↓
AI 越用越懂你
```

---

## 核心能力

### 1. 从历史对话中学习

即使你的对话已经被 `/compact` 压缩过，Memory Local 也能从 JSONL 日志文件中恢复完整历史。

**它记得你说过的每一句话。**

### 2. 自动分类

不是把所有信息塞进一个文件，而是按领域分类：

```
~/.claude/preferences/
├── 商业/          # 你谈商业模式时的思维
├── 模型建构/      # 你谈 AI 架构时的偏好
├── 项目开发/      # 你写代码时的习惯
└── 通用/          # 你的交流风格
```

**不同领域，不同记忆。**

### 3. 错误纠正升级

当 AI 犯同一个错误 3 次，自动升级为永久规则。

**它不会再犯同样的错。**

### 4. 按需加载

不一次性加载所有记忆。聊商业时加载商业偏好，写代码时加载开发习惯。

**上下文永远清爽。**

---

## 触发方式

**主动触发：**
- 「记录」
- 「请记录」
- 「学习我的习惯」

**自动触发：**
- 当你说「不对」「应该是」「你错了」时，自动记录纠正

---

## 安装

```bash
# 一行命令安装
mkdir -p ~/.claude/skills/memory-recorder && curl -o ~/.claude/skills/memory-recorder/SKILL.md https://raw.githubusercontent.com/leileigwl/memory_local/main/SKILL.md
```

## 使用场景

| 你是... | Memory Local 帮你... |
|---------|---------------------|
| 领域专家 | 让 AI 用你的专业视角思考 |
| 长期项目开发者 | AI 记住项目背景，不用重复说明 |

---

## 安全设计

- **只读核心配置**：不修改 CLAUDE.md
- **隔离存储**：只写入 `~/.claude/preferences/` 目录
- **本地优先**：所有数据在你本地，不上传任何地方

---

## 技术细节

- **语言**：Markdown + 自然语言指令
- **存储**：本地文件系统
- **依赖**：Claude Code CLI
- **兼容性**：支持 `/compact` 压缩后的 session 恢复
